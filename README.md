# SipLine HelloWorld Plugin

A simple demonstration plugin for **SipLine Softphone**.

## Purpose
This plugin serves as a learning resource for the SipLine SDK. It demonstrates:
*   Implementing the ISipLinePlugin interface.
*   Adding a custom button to the sidebar toolbar.
*   Displaying snackbar notifications.
*   Subscribing to SIP call events (Incoming, Outgoing, Ended).

## Getting Started
1. Clone this repository.
2. Ensure you have the SipLine.Plugin.Sdk NuGet package referenced.
3. Build and copy the DLL to your SipLine plugins/ folder.
